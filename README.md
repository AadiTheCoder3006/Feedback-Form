<h1>Event Hub Feedback Form 📋</h1>
<br>
A simple form built while learning HTML and basic web development.
This project is a part of my "Event Hub" platform and was created as a practice project to understand how web forms are structured and styled.
<br>
<br>
<h2>📌 Features</h2>
<br>

* Clean and modern UI
* Various input fields (text, email, number)
* Dropdown selection menu
* Radio buttons and checkboxes for user feedback
* Textarea for queries and suggestions
<br>
<br>

<h2>🛠️ Technologies Used</h2>
<br>
* HTML5<br>
* CSS3
<br>
<br>
<h2>📚 What I Learned</h2>
<br>
While building this project I practiced:
* Structuring pages using semantic HTML
* Using labels correctly and linking them to inputs
* Implementing form constraints (`required`, `min`, `max`)
* Working with different form elements like `select`, `textarea`, `radio`, and `checkbox`
<br>
